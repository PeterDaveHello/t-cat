# t-cat trace cli

Tiny script to help query package status from [t-cat](https://www.t-cat.com.tw/)

## dependencies

(All the tools can be installed via Debian/Ubuntu's apt tool)

- column
- grep
- html2text
- sh
- tail
- wget
- xargs

## usage

```sh
./t-cat <package number 1> <package number 2> <package number 3>
```
